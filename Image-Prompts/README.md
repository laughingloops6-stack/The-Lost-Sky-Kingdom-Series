# Image Prompts

## Purpose

This folder contains all AI image prompts used to create artwork for The Lost Sky Kingdom.

---

## Categories

- Characters
- Environments
- Creatures
- Weapons
- Magic Effects
- Buildings
- Vehicles
- Backgrounds
- Posters
- Thumbnails

---

## Prompt Format

Each prompt should include:

- Subject
- Environment
- Lighting
- Camera Angle
- Art Style
- Color Palette
- Quality Details

---

## Example Prompt

A young fantasy hero standing on a floating island above the clouds, golden sunrise, cinematic lighting, ultra detailed, anime style, 8K, vibrant colors.
